# LanScan Software — Portable Local Network Discovery Tool

## About LanScan Software

LanScan Software provides comprehensive local network scanning to discover all active devices on your subnet(s). LanScan Software performs ARP scans, ICMP ping sweeps, TCP port detection (common ports: SSH, HTTP, HTTPS, RDP, FTP), reverse DNS lookups, and service identification (Bonjour/mDNS, UPnP, NetBIOS, SMB). LanScan Software retrieves MAC addresses, manufacturer OUI lookup (vendor by first 3 octets), hostnames via reverse DNS and NetBIOS, and supports wake‑on‑LAN packet sending. LanScan Software exports scan results to CSV, HTML, JSON, filters by device type or status, and schedules periodic scans. LanScan Software integrates simple customizable IP range interface and command line automation directly into the tool. LanScan Software enables network administrators, IT support specialists, security analysts, and home users to understand what devices are connected to their networks, detect unauthorized devices, and troubleshoot connectivity. LanScan Software runs on portable storage, USB drives, and local user folders. LanScan Software is trusted by network administrators, IT support staff, security analysts, home network enthusiasts, and small business owners worldwide.
